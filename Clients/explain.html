使用されている技術

WebSocket

WebSocketは、WebブラウザとWebサーバー間で双方向通信を行うための技術です。
この技術を使用することで、リアルタイムでの情報のやり取りが可能になります。

WebSocket

Node.js

Node.jsは、JavaScriptのランタイム環境です。
このランタイム環境を使用することで、JavaScriptをサーバーサイドで実行することができます。
今回はtailwind cssをコンパイルするのに使いました

Tailwind CSS

Tailwind CSSは、CSSフレームワークの一つです。
このフレームワークを使用することで、webページのデザインするファイル(css)を書かなくても
htmlファイルだけでデザインすることができます

Tailwind CSS

Deno

Denoは、JavaScript/TypeScriptのランタイム環境です。
Node.jsのようなJavaScript/TypeScriptのランタイム環境で、TypeScriptやJSXをそのまま動かせることや
urlでインポートできたり、ESmoduleのみをサポートしていることが特徴です

Deno

Fresh

Freshは、Denoのフレームワークの一つです。 このフレームワークを使用することで、簡単にWebアプリケーションを作成することができます。
今回はapiとデータベースの通信やメッセージの処理に使用しましたがwebページのレンダリングなど様々な
用途に使用することができます。

Fresh

JavaScript

JavaScriptは、Webページを動的にするためのプログラミング言語です。
この言語を使用することで、Webページの動的な操作やアニメーションを行うことができます。
またapiと通信することによって新しいデータを取得することもできます

JavaScript

TypeScript

TypeScriptは、JavaScriptのスーパーセットであり、JavaScriptに型を追加したプログラミング言語です。
この言語を使用することで、JavaScriptのコードをより安全に書くことができます。
また、JavaScriptのコードをそのままTypeScriptのコードに変換することもできます

TypeScript

Git

Gitは、分散型バージョン管理システムです。
このシステムを使用することで、複数の開発者が同じプロジェクトを同時に開発することができます。
また、バージョン管理を行うことで、過去のコードに戻すことも可能です

Git このサイトのGitリポジトリー

mongoDB | mongoose

mongoDBは、NoSQLデータベースの一つです。
JSON形式のデータを保存することができ、
比較的高速でMySQLよりも柔軟なのが特徴です。
また、mongooseは、mongoDBを操作するためのライブラリです。

mongoDB mongoose

html

htmlは、Webページの構造を定義するためのマークアップ言語です。
この言語を使用することで、Webページの構造を定義することができます。
また、CSSやJavaScriptと組み合わせることで、Webページをデザインすることができます
正式名称はHyperText Markup Languageです

html
